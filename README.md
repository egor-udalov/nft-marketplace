## Файловая структура

**Базовая структура проекта**

```
└── fonts/ --------------------- # Шрифты
    ├── RobotoRegular.woff2         
└── images/ -------------------- # Картинки
    └── icons/
        ├── instagram_icon.svg                    
    ├── logo.jpg   
└── scripts -------------------- # JS файлы
└── styles/ -------------------- # CSS файлы
    └── components/ ------------ # Стили компонентов
        ├── button.css            
    └── sections/ -------------- # Стили для секций
        ├── footer.css            
        ├── header.css            
    ├── base-styles.css ------- # Базовые стили для всей страницы
    ├── fonts.css -------------- # Подключение шрифтов
    ├── reset.css -------------- # Сброс стилей
    ├── vars.css --------------- # CSS-переменные
├── index.html ----------------- # HTML

```
